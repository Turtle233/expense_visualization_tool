## 1. 总览

此 app 是一款基于 Qt6 (QML / Qt Quick Application) 框架开发的跨平台费用可视化工具。主要用于帮助用户将其支出按时间维度计算其日均成本，并以动态图表形式将其可视化。

相比传统只记录“总价”的记账方式，本应用通过时间分摊 + 数据可视化，提升用户对大额消费的感知力，并辅助做出更理性的消费决策 。

## 2. 技术栈

- Qt 6（跨平台框架）
- QML（UI设计）
- C++（核心逻辑）
- JSON（配置与数据存储）

## 4. 功能（功能性需求）

- 支持添加和管理多达5000条开支
- 支持数据排序（按时间 / 金额）
- 日均开支到周开支、月开支与年开支的转换计算
- 可拖动交互的图表，实现精确查看
- 多语言翻译热切换
- 多货币切换系统
- 深色 / 浅色卡片色切换
- 适配不同版本的客制化安卓系统
- 自定义实现 DateTimePicker（为跨平台开发做好准备）

## 5. 非功能性需求

### 🚀 性能

- 列表数据基于QListView + QJsonDocument，拥有百万级数据流的处理上限
- 精确到六位小数的货币切换
- 为所有可能的过渡提供了自定义动画效果，减少对安卓原生平台的动画依赖

### 👤 易用性

- 顶部、底部导航栏功能分离，卡片式列表简约高效
- 动画设计为移动端触控操作而生
- 新建项目提供自动导航至项目位置 + 红色边框反馈
- 输入错误提供底部信息框反馈
- 支持主题颜色切换
- 动态 DPI 自适应布局

### 🧩 可维护性

本项目设计旨在降低耦合性、提高内聚性

- UI 与逻辑分离：QML负责前端UI，C++负责后端逻辑，通过 Signal Slot 连接
- 隔离的导航机制 (edit mode / setting mode)
- 高度模块化 UI 设计 (Main -> Page -> Components)
- 核心功能分类 (calculation, currency, graph, items, language, sorting)
- 配置文件（JSON）独立存储

### 🔒 可靠性

- 添加与编辑对话框提供多层输入合法性校验
- 通过边界测试
- 安卓12+的沙箱环境将数据隔离，用户无法访问与操作
- DateTimePicker等组件单独手写解耦合，适配未来多平台开发
